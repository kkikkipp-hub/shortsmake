# ShortsMake — 롱폼 → 숏폼 자동 변환기 PRD

## 1. 프로젝트 개요

**한 줄 요약:** 저작권 프리 롱폼 영상 URL을 입력하면 AI가 분석하여 여러 개의 숏폼 영상을 자동 생성하는 웹 서비스

**핵심 가치:**
- 영상 편집 경험 없이도 고품질 숏폼 콘텐츠 대량 생산
- AI 기반 하이라이트 구간 자동 추출
- 자막 편집 → Edge TTS 음성 합성 → 영상 효과 자동 적용

---

## 2. 기술 스택

**프론트엔드**
- Vite + React + TypeScript
- Cloudflare Pages 배포

**백엔드 (2-tier)**
- Cloudflare Pages Functions (Hono) → D1/R2 접근, 메타데이터 API
- Python FastAPI (별도 프로세스) → 영상 처리 (FFmpeg, Whisper, TTS)

**데이터베이스:** Cloudflare D1 (SQLite)
**파일 저장:** Cloudflare R2 (S3 호환)
**영상 처리:** FFmpeg + faster-whisper + edge-tts
**AI 분석:** OpenAI GPT-4o API

---

## 3. 주요 기능

### Phase 1: 핵심 파이프라인

**F1. 영상 입력 & 다운로드**
- YouTube URL 입력 (저작권 프리 영상)
- yt-dlp로 영상 다운로드 (최대 1080p)
- 영상 메타데이터 표시 (제목, 길이, 해상도)
- 로컬 파일 업로드 지원
- 다운로드 완료 → R2에 원본 저장

**F2. AI 영상 분석 & 구간 추출**
- faster-whisper STT로 전체 영상 자막 자동 생성
- GPT-4o가 자막 분석하여 숏폼 후보 구간 자동 추천
- 각 구간별: 시작/끝 시간, 추천 이유, 매력도 점수
- 숏폼 길이 설정: 15초 / 30초 / 60초 / 사용자 지정

**F3. 자막 편집기**
- 추출된 자막 타임라인 표시
- 자막 텍스트 수정 (오타, 문맥 교정)
- 자막 타이밍 조정 (시작/끝 시간)
- 자막 추가/삭제
- 자막 스타일 설정 (폰트, 크기, 색상, 위치, 배경)

**F4. TTS 음성 합성**
- Edge TTS 한국어 음성 선택 (4종)
- 자막 텍스트 기반 TTS 생성 + 자동 싱크
- 원본 오디오 vs TTS 선택 (또는 혼합)
- TTS 속도/피치 조절

**F5. 영상 효과 & 최종 변환**
- 가로(16:9) → 세로(9:16) 자동 변환
- 영상 효과 프리셋:
  - 줌인/줌아웃 (Ken Burns 효과)
  - 페이드 인/아웃
  - 분할화면 (PiP, 2분할, 3분할)
  - 텍스트 팝업 애니메이션
  - 배경 블러 + 중앙 원본 (세로 변환용)
  - 색보정 필터 (밝게, 빈티지, 시네마틱)
- ASS 자막 오버레이 렌더링
- 최종 MP4 출력 (1080x1920, 9:16)
- 일괄 변환 → R2에 저장 → 다운로드 링크

### Phase 2: 확장 기능

**F6.** 변환 전 실시간 프리뷰
**F7.** 효과 템플릿 시스템
**F8.** 생성 이력 관리 + 재편집

## 4. 사용자 플로우

```
[1] URL 입력 또는 영상 업로드
         ↓
[2] 영상 다운로드 & 분석 (프로그레스 바)
    - Python서버: yt-dlp 다운로드 → R2 저장
    - Python서버: faster-whisper STT → 자막 생성
    - GPT-4o: 하이라이트 구간 자동 추출
    - D1: 메타데이터 + 자막 + 구간 저장
         ↓
[3] 구간 선택 화면
    - AI 추천 구간 목록 (카드형)
    - 각 구간 미리보기 썸네일
    - 숏폼 길이 설정 (15/30/60초/커스텀)
    - 원하는 구간 체크박스 선택
         ↓
[4] 자막 편집 화면 (선택한 구간별)
    - 영상 플레이어 + 자막 리스트
    - 텍스트 수정, 타이밍 조정
    - 자막 스타일 설정
         ↓
[5] 효과 & TTS 설정
    - 영상 효과 프리셋 선택
    - TTS 음성 선택 (또는 원본 오디오 유지)
    - 분할화면/줌 등 효과 옵션
         ↓
[6] 최종 변환 & 다운로드
    - Python서버: FFmpeg 렌더링 (진행률 SSE)
    - 완성본 → R2 저장
    - 개별/전체 다운로드 (R2 presigned URL)
```
