# ShortsMake — UI 화면 설계

## 페이지 구조

```
/ (HomePage)         — URL 입력 + 최근 작업 목록
/analyze/:id         — 분석 진행 → 구간 선택
/edit/:id            — 자막 편집 + 효과 설정 + TTS
/render/:id          — 렌더링 진행 + 다운로드
```

## 컴포넌트 목록

**공통**
- Header (로고 + 네비게이션 breadcrumb)
- ProgressBar (다단계 프로그레스)

**HomePage**
- UrlInput (URL 입력 + 유효성 검사)
- FileUpload (드래그앤드롭)
- RecentWorkList (최근 작업 카드)

**AnalyzePage**
- AnalysisProgress (3단계 진행률)
- SegmentCard (구간 카드: 체크박스 + 미리보기 + 정보)
- DurationSelector (숏폼 길이 선택)

**EditPage**
- VideoPlayer (HTML5 + 자막 오버레이 + 커스텀 컨트롤)
- SubtitleEditor (자막 리스트 + 편집)
- SubtitleRow (개별 자막: 시간 + 텍스트)
- SubtitleStylePanel (폰트/크기/색상/위치)
- EffectSelector (효과 프리셋 선택)
- TtsPanel (음성 선택 + 속도 + 미리듣기)

**RenderPage**
- RenderProgress (실시간 진행률)
- RenderResult (완료 카드: 미리보기 + 다운로드)
